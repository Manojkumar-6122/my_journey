# AI Skill - MCP Learning

## What is a Skill?

A Skill is a reusable set of instructions that teaches an AI how to perform a specific task. Instead of writing the same prompt repeatedly, we can create a Skill once and use it whenever needed.
Just like a function in javascript which can be rusable.

## Why are Skills useful?

- They save time.
- They provide consistent responses.
- They make AI agents easier to maintain.
- They reduce repetitive prompts.

## Example

A Git Skill can help an AI:
- Create branches
- Write commit messages
- Explain Git commands
- Help create Pull Requests

A Java Skill can help:
- Explain Spring Boot code
- Debug errors
- Generate unit tests
- Improve code quality
