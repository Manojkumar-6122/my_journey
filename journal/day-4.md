## What I Learned Today

Today I learned about AI Skills and the Model Context Protocol (MCP). I understood that a Skill is a reusable set of instructions that helps an AI agent perform a specific task consistently. I also learned that MCP is a standard protocol that allows AI agents to communicate with external tools and services. I created my own `SKILL.md` file, wrote my understanding of MCP, organized my repository by separating learnings, exercises, and journals, and updated the README file.

I also learned how to use interactive rebase in Git to edit commit messages and understood the importance of using conventional commit types like `feat:` and `docs:`.
