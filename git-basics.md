# Git Basics



## Five Common Git Commands

### 1. git init
Creates a new Git repository in the current project folder.



### 2. git status
Shows the current status of the repository, including modified, staged, and untracked files.


### 3. git add
Adds files to the staging area so they are ready to be committed.

### 4. git commit
Saves the staged changes with a descriptive message.

### 5. git push
Uploads the committed changes from your local repository to GitHub.
