# Day-04 MCP Learning

## What is a Skill?

A Skill is a reusable instruction or capability that teaches an AI how to perform a particular task. It helps the AI give consistent and accurate responses without repeating the same instructions every time.

---

## What is MCP (Model Context Protocol)?

Model Context Protocol (MCP) is a standard communication protocol that allows AI models to connect with external tools, applications, and services. It enables AI agents to exchange information and perform actions securely using a common interface.

---

## How our agents connect to the Ticket Agent using MCP

Our AI agents communicate with the Ticket Agent using MCP. The agent sends requests through the MCP protocol, and the Ticket Agent processes those requests and returns the required information. This standard communication makes it easy for different AI agents and tools to work together.

---

## What I Learned

- Skills make AI agents reusable and organized.
- MCP provides a standard way for AI to communicate with external systems.
- AI agents can collaborate through MCP instead of using separate integrations.
- Skills and MCP improve the efficiency and scalability of AI applications.
