# Day 03 - Claude Code Learnings

## What is Claude Code?

Claude Code is an AI coding assistant that works inside the terminal. Unlike traditional autocomplete tools, it can understand an entire project, search files, edit code, run commands, and help build complete features.

---

## What is Plan Mode?

Plan mode means asking Claude to think before writing code.

Instead of immediately generating code, Claude first:
- Understands the problem
- Creates a plan
- Explains the approach
- Waits for approval

This usually results in better code and fewer mistakes.

---

## Why review every diff?

Every code change should be reviewed before accepting it because AI can misunderstand requirements or introduce bugs.

Reviewing diffs helps ensure:
- Code is correct
- No unnecessary changes
- Coding standards are followed
- No accidental deletions

---

## #1 Danger

The biggest mistake is blindly accepting AI-generated code.

Always:
- Read the code
- Test it
- Understand it
- Verify it works

AI is a powerful assistant but not a replacement for developer judgment.
